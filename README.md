# Mongo & Node
Conectar Mongo Atlas con nuestro servidor en NodeJs

## Getting Started
Clonar este Repo:
```
git clone https://github.com/ArturVargas/08_Mongo.git
```
Ejecutar en el path del proyecto
```
npm install --save
```
para instalar todas las dependencias.
